# DeadLeaf 📚

A quiet place to read stories, books, and manga — without downloads, without noise, without distraction.

![DeadLeaf Logo](images/Deadleaf.png)

## ✨ Features

- **📖 Clean Reading Experience** - Minimalist interface that puts your content first
- **🎨 Three Beautiful Themes** - Dark, Night, and Sepia modes for comfortable reading anytime
- **📑 Smart Bookmarks** - Save your favorite moments and return to them instantly
- **📊 Reading Stats & Goals** - Track your progress and build reading streaks
- **🔐 User Accounts** - Sync your progress across devices
- **🌐 Social Login** - Sign in with Google or GitHub
- **📱 Fully Responsive** - Works perfectly on desktop, tablet, and mobile

## 🛠️ Tech Stack

- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **Backend**: Supabase (PostgreSQL, Authentication, Storage)
- **Authentication**: Supabase Auth + Google OAuth + GitHub OAuth
- **Hosting**: Ready for Vercel/Netlify deployment
- **Icons**: Font Awesome 6

## 📋 Prerequisites

Before you begin, ensure you have:
- A [Supabase](https://supabase.com) account (free tier works perfectly)
- A [Google Cloud Console](https://console.cloud.google.com) project (for Google OAuth)
- A [GitHub](https://github.com) account (for GitHub OAuth and hosting)
