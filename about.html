The Story

DeadLeaf began as a personal project — a reaction to the noisy reading platforms that interrupt the very thing they're meant to provide: peace.

As a writer myself, I wanted a space where stories could be experienced without distractions. Where the interface fades away and only the words remain. Where reading feels like it should — quiet, personal, uninterrupted.

Every line of code, every design decision, every feature is built with one philosophy: respect the reader's attention.

One Person, Many Hats

DeadLeaf isn't a team — it's just me. Everything you see is built by one pair of hands:

  • Designer — Every pixel, every animation
  • Developer — Frontend, backend, all the code
  • Support — If you need help, you're talking to me
  • Management — Planning, organizing, dreaming
  • Writer — Movie scripts, anime concepts, stories
  • 3D Modeler — Creating worlds in three dimensions
  • Game Developer — Interactive storytelling

It's a lot of hats, but someone's got to wear them.
